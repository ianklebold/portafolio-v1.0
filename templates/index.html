Proyecto #1 : FrontEnd para inmobiliaria DOMUS

Domus es una inmobiliaria la cual requeria implementar una aplicacion web para llevar al dia las citas, trabajos y eventos de la organizacion. A su vez necesitaban ayuda para contactar con clientes y poner al dia a los agentes inmobiliarios con sus actividades.

En este proyecto me encargué del FrontEnd de la pagina web, la misma fue realizada con HTML, CSS y JavaScript, apoyandonos del framework Bootstrap 5.

Actualmente el proyecto en frontend sigue recibiendo mejoras pendientes y a su vez se encuentra en desarrollo el Backend con la tecnologia de Spring MVC.

Proyecto #3 : Galaxian

Galaxian es un videojuego creado con un grupo de estudiantes de la universidad a la cual estudio. El videojuego se trata de eliminar tantos aliens como sea posible, sumando la mayor cantidad de puntos intentado superar records. El juego contaba con menu de opciones, tablero del juego, dificultad que incrementaba nivel a nivel y una tabla de los mejoes puntajes.

Este proyecto lo hicimos ni mas ni menos que con el mitico lenguaje programacion orientada a objetos Smalltalk, aprovechando de las herramientas Morph que proporcionaba el lenguaje.

Para la ejecucion del juego fue necesario tener la IDE Pharo la cual nos permite correr el lenguaje Smalltalk.

Proyecto #4 : Red social de apuntes

La red social de apuntes fue un mini proyecto realizado en el verano del 2021. Nacio como propuesta para poder compartir apuntes, notas, modelos de examenes, actividades, ejercicios de la facultad. La idea era compartir conocimiento a traves de una red social de apuntes, donde uno podia loguearse, configurar su perfil y compartir sus conocimientos

Este proyecto fue realizado con Python en el framework Django, utilizando solamente HTML y CSS

Actualmente este proyecto fue suspendido por falta de tiempo, espero poder retomarlo, mejorando su codigo en tanto funcionalidad y legibilidad del codigo implica.
